Light and Sound Game

Rules: Each round, a sequence of tiles will light up and make a sound. The game starts with 1 tile and increments all the way up to 8. You win the game if you can guess all the patterns correctly! You get 20 seconds each round, and a total of 3 tries!

Note: In the hard mode, only the correct pattern will light up, and all the other blocks will disappear. This prevents you from retaining spatial memory, which will make the game harder.

Game Mode: Normal

Time Left: 20
